Вадим Монада

Дневник тренировок

День 1

Итак, поехал! Cегодня провел первую тренировку из серии: тренируюсь только в упражнении бой с тенью до первого боксерского поединка. Поработал сегодня 6 раундов по 3 минуты с отдыхом между раундами в 1 минуту. После чего поработал еще один раунд в три минуты, отснял видео для своего блога, где применял все элементы которые я прорабатывал в основной тренировке (6рд/3мин/1мин).

В целом, сегодняшняя тренировка прошла по следующему принципу:

  • Раунд 1 - отрабатываю спонтанные атаки начиная с левого прямого удара.
  • Раунд 2 - отрабатываю спонтанные атаки начиная с правого прямого удара.
  • Раунд 3 - отрабатываю спонтанные атаки начиная с левого бокового удара.
  • Раунд 4 - отрабатываю спонтанные атаки начиная с правого бокового удара.
  • Раунд 5 - отрабатываю спонтанные атаки начиная с левого снизу удра.
  • Раунд 6 - отрабатываю спонтанные атаки начиная с правого снизу удара.
  • В завершении отрабатываю еще один раунд в три минуты для видео.

Достаточно лишь только тренировки бой с тенью?

Решил провести эксперимент, следуя теории что тренировки бой с тенью сами по себе являются самодостаточным упражнением, для того чтобы быть способным выиграть боксерский поединок. С сегодняшнего дня, буду практиковаться только в тренировках бой с тенью. Никакого бега, никаких отжиманий, приседаний, скручиваний и подтягиваний. Само по себе разумеется что никакого боксерского мешка и работы в парах. Через пять месяцев хочу провести боксерский поединок, и исходя из его исхода сделаю выводы на сколько эффективна сама по себе тренировка бой с тенью. Тренироваться я планирую от трех до семи раз в неделю. По объему и интенсивности каждая тренировка будет разной. Регулировать интенсивность и объемы буду по самочувствию. У меня есть много идей как проводить тренировки, я люблю создавать различные вариации тренировок бой с тенью.
